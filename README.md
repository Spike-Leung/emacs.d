# Installation

To install, clone this repo to `~/.emacs.d`, i.e. ensure that the `init.el` contained in this repo ends up at `~/.emacs.d/init.el`:

```bash
git clone --recurse-submodules -j8 git@github.com:Spike-Leung/emacs.d.git ~/.emacs.d
```
